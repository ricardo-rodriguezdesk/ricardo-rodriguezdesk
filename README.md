## Sobre mi

Soy estudiante de Ingenieria en Sistemas Computacionales y desarrollador de software en activo. Me gusta construir aplicaciones que resuelvan problemas reales, no solo ejercicios de clase: desarrolle y mantengo un sistema ERP en produccion para una empresa real, construi una app movil en un hackathon, y participe como desarrollador de backend y ML en un proyecto de prediccion de datos.

- **Desarrollo de software** — Construyo aplicaciones de escritorio, web y moviles, con foco en codigo mantenible y funcionalidad que de verdad se use.
- **Full Stack** — Trabajo desde la interfaz hasta la base de datos: Java/JavaFX, C#/WPF, Python, Node.js y React Native.
- **AI / ML** — En aprendizaje activo; desarrolle modelos de prediccion y APIs para un proyecto de datos.
- **Mentalidad de producto** — Me interesa entender el problema del usuario antes de escribir codigo.

> **Abierto a:** practicas, oportunidades junior de desarrollo de software, y colaboracion en proyectos.

## Proyectos destacados

<details>
<summary><b>ZAER Company ERP — Sistema de Gestion Empresarial (C# / .NET)</b></summary>

Sistema ERP de escritorio en produccion para una empresa real. Version 2.0 del sistema, reconstruida desde cero en C# / WPF con una arquitectura mas robusta y modular que la primera version en Java.

| Atributo | Detalle |
| :--- | :--- |

</details>

<details>
<summary><b>BasuraApp Fenix — App Movil de Recoleccion de Residuos (Hackathon)</b></summary>

<br/>

App movil que notifica a la ciudadania de Celaya la hora aproximada del camion recolector de basura, bajo un principio de Privacidad por Diseno. Desarrollada durante el hackathon HackOnLinces 2026.

| Atributo | Detalle |
| :--- | :--- |
| **Mi rol** | Desarrollo tecnico completo (frontend, backend, base de datos, tiempo real) |
| **Stack** | React Native · Expo · Python · FastAPI · PostgreSQL · WebSockets · JWT |
| **Conceptos** | Tiempo real (WebSockets) · RBAC · geolocalizacion (Haversine) · APK nativo (EAS Build) |
| **Datos** | 220 colonias reales de Celaya mapeadas a 15 rutas |
| **Contexto** | Hackathon HackOnLinces 2026 — equipo Fenix |
| **Repositorio** | [github.com/ricardo-rodriguezdesk/BasuraApp-Fenix](https://github.com/ricardo-rodriguezdesk/BasuraApp-Fenix) |

Implemente el sistema completo de extremo a extremo: frontend en React Native, backend en FastAPI, ETA en tiempo real via WebSockets y mapeo geoespacial de domicilios a rutas con el algoritmo Haversine.

</details>

<details>
<summary><b>GasolinaIQ — Prediccion de Precios de Gasolina (Backend + ML)</b></summary>

<br/>

Plataforma de analisis y prediccion de precios de gasolina en Mexico con datos publicos de la CRE, BANXICO y EIA. Proyecto de equipo para la materia de Big Data.

> **Nota:** mis contribuciones a este proyecto estan bajo mi cuenta institucional [@Rizzko557](https://github.com/Rizzko557).

| Atributo | Detalle |
| :--- | :--- |
| **Mi rol** | Desarrollador principal de Backend y ML |
| **Mi aporte** | API REST en Node.js · modelo de prediccion con Prophet en todos los endpoints · busqueda por cercania (GPS) · score calidad-precio · carga masiva de 13,649 estaciones por coordenadas · scripts ETL · dashboard servido desde API · Dockerfile |
| **Stack** | Node.js · Express · Python · Prophet · MongoDB · Docker · AWS S3 |
| **Equipo** | 5 integrantes |
| **Repositorio** | [github.com/Victorml55/GasIQ](https://github.com/Victorml55/GasIQ) |

Desarrolle la API REST completa en Node.js, integre el modelo predictivo de Prophet en todos los endpoints, implemente la busqueda por cercania por GPS, el score calidad-precio y la carga masiva de mas de 13,000 estaciones mapeadas por coordenadas.

</details>
